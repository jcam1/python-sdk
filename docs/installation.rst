インストール
========

Poetry を使用したインストール
-------------------------

.. code-block:: bash

   $ poetry add jpyc-sdk-python

Pip を使用したインストール
-----------------------

.. code-block:: bash

   $ pip install jpyc-sdk-python

依存関係
-------

JPYC Python SDKには以下の依存関係があります：

- Python >=3.11
- web3 >=7.9.0,<8.0.0
- pydantic >=2.10.6
- typing-extensions >=4.12.2
- python-dotenv >=1.0.1
- hexbytes >=1.2.0
